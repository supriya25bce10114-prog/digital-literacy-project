## Project Overview

This repository contains my Digital Literacy Portfolio created as part of the CSE0001 course at VIT Bhopal University. The project demonstrates essential digital literacy skills including infographic creation, professional profile setup, coding platform practice, email etiquette, collaboration tools usage, and cybercrime awareness.

---

## Repository Structure

### Task 1 – Digital Literacy Awareness Infographic
Created an infographic explaining digital literacy concepts, useful digital tools for students, safe internet practices, professional online presence and email etiquette.

### Task 2 – Student Digital Portfolio
Created professional profiles on platforms like GitHub, LinkedIn, and HackerRank to build a strong digital presence.

### Task 3 – Coding & Collaboration Platforms
Completed a beginner-level coding challenge and created a Digital Literacy Awareness Quiz using Google Forms.

### Task 4 – Professional Email & Social Media Etiquette
Drafted professional emails and prepared a checklist of responsible social media usage for students.

### Task 5 – Cybercrime Awareness Case Study
Prepared a case study on cybercrime and created a prevention checklist for safe online practices.
